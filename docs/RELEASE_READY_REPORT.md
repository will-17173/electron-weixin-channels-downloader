## 📋 项目概览

### 🎯 核心功能
- **智能监控**: 自动监控微信视频号页面，实时捕获视频数据
- **批量下载**: 支持多任务并发下载，进度实时显示
- **代理服务**: 内置HTTP/HTTPS代理服务器，透明流量拦截
- **跨平台支持**: Windows、macOS、Linux 三大平台
- **现代界面**: 基于 Electron + Vue 3 的现代化用户界面

### 🛡️ 安全特性
- **本地处理**: 所有数据处理均在本地完成

- **权限控制**: 最小权限原则
